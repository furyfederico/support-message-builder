# Support Message Builder

A simple static tool to generate clear support messages for payout issues, KYC checks, and bonus disputes.

## What this tool does

- Lets the user choose a case type: payout, KYC, or bonus dispute.
- Collects key details (username, short summary, full details).
- Generates a structured English message that can be sent to support.

## Project files

- `index.html` — UI markup.
- `styles.css` — visual styles.
- `app.js` — message builder logic.

## How to run (GitHub only, no terminal)

1. Create a new GitHub repository named `support-message-builder`.
2. Click **Add file → Create new file** and add:
   - `index.html`
   - `styles.css`
   - `app.js`
3. Click **Commit changes** for each file.
4. Open **Settings → Pages**.
5. Under **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: `main`
   - Folder: `/ (root)`
6. Click **Save**.
7. Wait 1–2 minutes and open the published URL, for example:
   `https://furyfederico.github.io/support-message-builder/`
